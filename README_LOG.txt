ИНСТРУКЦИЯ ПО НАСТРОЙКЕ ЛОГИРОВАНИЯ В TXT ФАЙЛ
================================================

1. Загрузите файл log.php на ваш VPS сервер в папку с вашим сайтом
   (например, в /var/www/React/ или где у вас находится сайт)

2. Убедитесь, что у PHP есть права на запись в эту папку:
   chmod 755 log.php
   chmod 777 .  (или дайте права на запись в папку)

3. Настройте URL во фронтенде:
   
   Создайте файл .env в корне проекта:
   VITE_LOG_URL=https://your-domain.com/log.php
   
   Или измените напрямую в src/App.tsx:
   const LOG_URL = 'https://your-domain.com/log.php';

4. Логи будут автоматически записываться в файл invites_log.txt
   в той же папке, где находится log.php

5. Для просмотра логов на сервере:
   cat invites_log.txt
   или
   tail -f invites_log.txt  (для просмотра в реальном времени)
