# Как получить токен бота и URL приложения

## 1. Как получить токен бота (BOT_TOKEN)

### Шаг 1: Создайте бота через @BotFather

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям:
   - Придумайте имя для бота (например: "My React Bot")
   - Придумайте username для бота (должен заканчиваться на `bot`, например: `my_react_bot`)
4. BotFather выдаст вам токен, который выглядит так:
   ```
   1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
   ```

### Шаг 2: Скопируйте токен

Скопируйте этот токен и вставьте в `config.py`:
```python
BOT_TOKEN = "1234567890:ABCdefGHIjklMNOpqrsTUVwxyz"
```

### Шаг 3: Настройте WebApp (важно!)

После создания бота нужно настроить мини-приложение:

1. Отправьте BotFather команду `/newapp`
2. Выберите вашего бота
3. Укажите название приложения
4. **Укажите URL вашего React приложения** (см. ниже, как его узнать)
5. Загрузите иконку (опционально)

Или используйте команду `/setmenubutton` для настройки кнопки меню.

## 2. Как узнать URL React приложения (WEB_APP_URL)

URL зависит от того, где у вас развернуто приложение.

### Вариант 1: Если приложение на вашем VPS

Если вы развернули приложение на VPS через веб-сервер (nginx/apache), URL будет:

```
https://ваш-домен.com
```

или

```
https://ваш-ip-адрес
```

**Пример:**
```python
WEB_APP_URL = "https://myapp.com"
# или
WEB_APP_URL = "https://123.456.789.0"
```

### Вариант 2: Если используете хостинг

Если используете хостинг (например, Vercel, Netlify, GitHub Pages), URL будет предоставлен хостингом:

**Vercel:**
```
https://your-project.vercel.app
```

**Netlify:**
```
https://your-project.netlify.app
```

**GitHub Pages:**
```
https://username.github.io/repository-name
```

### Вариант 3: Локальная разработка (только для теста)

Для локальной разработки можно использовать ngrok или подобные сервисы:

```bash
# Установите ngrok
# Запустите ваш React dev сервер
npm run dev  # обычно на http://localhost:5173

# В другом терминале запустите ngrok
ngrok http 5173
```

Ngrok выдаст вам URL вида:
```
https://abc123.ngrok.io
```

⚠️ **Внимание:** Для продакшена используйте реальный домен, не ngrok!

## 3. Как проверить, что URL правильный

1. Откройте URL в браузере
2. Должна открыться ваша React приложение
3. Убедитесь, что приложение работает через HTTPS (Telegram требует HTTPS для WebApp)

---

## 4. Пример заполненного config.py

```python
# Конфигурация бота
# Заполните все необходимые данные

# Токен бота (получите у @BotFather)
BOT_TOKEN = "1234567890:ABCdefGHIjklMNOpqrsTUVwxyz"

# URL вашего мини-приложения (React приложение)
WEB_APP_URL = "https://myapp.com"

# Дополнительные настройки (опционально)
BOT_NAME = "React Bot"
ADMIN_IDS = []  # Список ID администраторов (если нужно)
```

## 5. Частые проблемы

### Проблема: "Invalid token"
- Проверьте, что токен скопирован полностью (без пробелов)
- Убедитесь, что используете токен от правильного бота

### Проблема: "WebApp не открывается"
- Убедитесь, что URL работает и доступен через HTTPS
- Проверьте, что URL указан в настройках бота через BotFather
- Убедитесь, что приложение собрано и развернуто (`npm run build`)

### Проблема: "CORS ошибка"
- Убедитесь, что приложение доступно по HTTPS
- Проверьте настройки веб-сервера (nginx/apache)
